# HR-ANALYTICS-EXCEL-PROJECT-DASHBAORD



Here is a quick overview of what this dashboard delivers:



Core KPI Tracking: Instantly monitors workforce metrics including total headcount (882), active staff (732), attrition count (150), and attrition rate (17.01%).



Educational \& Role Breakdown: Analyzes employee turnover across qualifications (Bachelor's, Master's, etc.) and specific job roles.



Departmental \& Age Insights: Pinpoints turnover trends across major departments (R\&D, Sales, HR) and key age brackets (highest in ages 25–34).



Demographic Patterns: Evaluates attrition variations based on marital status and gender split.



Interactive Filtering: Built with dynamic slicers (Gender, Education Field, Department) for real-time deep dives.



Data-Driven HR Decisions: Empowers leadership to identify flight risks and optimize retention strategies.
